Папка для ONNX-модели Real-ESRGAN.

Положите сюда файл с именем:
  • Real-ESRGAN-General-x4v3.onnx

Сохранение сразу в эту папку при конвертации из .pth:
  Запустите convert_esrgan_to_onnx.bat и передайте путь к .pth файлу.
  Если esrganONNX.py лежит не в C:\Users\nkova\, откройте .bat и измените путь к скрипту.

Где взять модель: https://github.com/xinntao/Real-ESRGAN или готовый ONNX (real-esrgan onnx download).
